About Us

We are an independent company that specializes in teaching people about programming. Our company places great focus on the programming that builds the websites. That is, we learn ours readers especially around HTML, CSS programming and JavaScript. Our website is called “WebBrain.com”. The website is two words, put together into one. “Web” stands for internet in English, and “Brain” is brain, then one must learn to program. In Danish, our website will be called "Internet brain", which we think is an appropriate name for a website that learns about website programming.
The website is free to use for everyone, however, there will be advertisements on the website, to make one came into the company. The user can remove this, by purchasing a Premium account, to remove all advertisements on the website. That way, our website gets a guaranteed income for the company.
