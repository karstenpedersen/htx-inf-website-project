Beginner CSS - Rounded Corners

Written by https://www.w3schools.com. Inserted by Lucas Engsbye.

CSS is the language we use to style a Web page. CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

Examples

With the CSS border-radius property, you can give any element "rounded corners".

  • Rounded corners for an element with a specified background color

    Code

    #rcorners1 { border-radius: 25px; background: #6200ff; padding: 20px; width: 200px; height: 150px; }

    Result

    Rounded corners!

  • Rounded corners for an element with a border

    Code

    #rcorners2 { border-radius: 25px; border: 2px solid #6200ff; padding: 20px; width: 200px; height: 150px; }

    Result

    Rounded corners!

  • Rounded corners for an element with a background image

    Code

    #rcorners3 { border-radius: 25px; background: url(../images/paper.jpg); background-size: contain; background-position: left top; background-repeat: repeat; padding: 20px; width: 200px; height: 150px; }

    Result

    Rounded corners!
