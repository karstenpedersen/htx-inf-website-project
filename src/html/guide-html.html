Beginner HTML

Written by https://www.w3schools.com. Inserted by Magnus Jens Ingemansen.

HTML is a standard language used in connection with the creation of websites. HTML is an abbreviation for Hyper Text Markup Language and is used to describe the structure of a website.

Elements

HTML consists of a number of different elements. These elements tell the browser how to display the content on the page.

Code

<tagname>Content should be...</tagname>

Headers

To make headlines or headers, one has to use and number and prioritize it according to which header is most important. In this case, the main header. Example:

Code

<h1>This is header 1</h1> <h2>This is header 2</h2> <h3>This is header 3</h3>

Result

This is header 1

This is header 2

This is header 3

Paragraphs

In these paragraphs or sections, all your body text will be in. They make them using the tag that marks where your paragraphs should be. Example:

Code

<p>This is a paragraph</p> <p>This is another paragraph</p>

Result

This is a paragraph

This is another paragraph

HTML document setup

HTML documents are where you encode HTML in. All HTML documents must start with a document type statement: . It starts with html declaration and ends with a html declaration. Finally, the visible parts of websites are inside the "body" of it.

Images

The HTML image tag is used to embed an image in a web page. Images are not technically inserted into a web page; images are linked to web pages. The image tag creates a holding space for the referenced image. The image tag is empty, it contains attributes only, and does not have a closing tag.
